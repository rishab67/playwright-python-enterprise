# PROJECT IDENTITY

## Project Name
Playwright Python Enterprise Framework

## Project Goal
Build a production-grade enterprise automation framework using Playwright and Python.

## Core Stack
- Playwright
- Python
- Pytest
- Page Object Model (POM)

## Planned Enterprise Features
- CI/CD Integration
- GitHub Actions
- Allure Reporting
- Parallel Execution
- API + UI Testing
- Docker Support
- Jira Integration
- Slack Notifications
- AI-assisted Reporting

## Engineering Principles
- Stability before expansion
- Reusable architecture
- Maintainable code
- Scalable framework design
- Enterprise-grade standards

## Critical Rules
- Do not move to unrelated features during unresolved debugging.
- Avoid duplicate implementations.
- Do not assume project state without verification.
- Prioritize framework stability before adding new integrations.

## Current Status
Framework actively under development and evolving toward enterprise-level implementation.
